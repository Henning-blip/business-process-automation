# Business Process Automation (n8n)

Dieses Repository enthält visuelle Workflows zur Automatisierung von Makler-Prozessen. 

## Business Case
Reduktion manueller Copy-Paste-Aufgaben zwischen CRM und Kommunikationskanälen durch Webhooks und API-Orchestrierung. Die effiziente Verwendung von HTML, wie im Artikel "Using Claude Code: The unreasonable effectiveness of HTML" erwähnt, wird hierbei eine zentrale Rolle spielen, um Integrationen zuCRM und.Communikationskanälen zu verbessern.

*(Hinweis: Sensible Endpunkte und Prompts wurden aus dem Workflow-Export entfernt.)*
